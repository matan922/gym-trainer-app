## Features

### Trainer System (Implemented)
- **Client Management**: Trainers can add, edit, delete, and view clients.  
- **Workout Management**: Trainers can create and manage workouts for clients.  
- **Session Management**: Trainers can schedule sessions, mark them as completed, or cancel them.  
- **Daily Overview**: Trainers can see who is training today and the total number of sessions.  
- **Client Activity Tracking**: See the last time a client trained.
